# User Flow — 성심당 스탬프 웹 (데모)

- 문서 버전: v1.0
- 기준 문서: PRD v1.1
- 표기 규칙: [화면] / (사용자 행동) / 《시스템 동작》

---

## 0. 화면 구성 요약

| 화면 ID | 화면명 | 진입 경로 |
|---|---|---|
| S1 | 메인 화면 | 최초 접속, 하단 탭 "홈" |
| S2 | 스탬프 적립 팝업 | 적립 발생 시 자동 노출 (S1 위에 모달) |
| S3 | 스탬프 내역 (달력) | 하단 탭 "스탬프 내역" |
| S4 | 마이페이지 | 하단 탭 "마이페이지" |

하단 내비게이션(홈 / 스탬프 내역 / 마이페이지)은 모든 화면에서 항상 노출되며, 탭 간 자유롭게 이동 가능하다.

---

## 1. 최초 방문 플로우 (닉네임 설정)

목적: 로그인 없이 닉네임만으로 시작

1. (사용자) 웹 주소로 접속
2. 《localStorage에 닉네임 존재 여부 확인》
3. **닉네임 없음(최초 방문)** →
   - [닉네임 설정 화면 또는 팝업] "성심당에서 부를 이름을 알려주세요"
   - (사용자) 닉네임 입력 (1~10자, 공백만 입력 불가)
   - 《유효성 검사 → 통과 시 localStorage 저장》
   - → [S1 메인 화면] "{닉네임}님, 안녕하세요" 노출
4. **닉네임 있음(재방문)** → 바로 [S1 메인 화면]

예외 처리
- 닉네임이 유효하지 않으면: 입력창 아래 오류 문구 노출, 저장 버튼 비활성화

---

## 2. 스탬프 적립 플로우 (핵심 플로우)

목적: 태깅 → 적립 → 확인의 핵심 경험 검증
※ 데모에서는 NFC 태깅 대신 시뮬레이션 트리거로 대체. 실제 서비스에서는 "NFC 태그 접촉 → 지점 URL 접속"이 1~2단계를 대신한다.

1. [S1 메인 화면] (사용자) 스탬프 적립 시뮬레이션 트리거 실행
2. 《적립 처리》
   - 지점 결정 (데모: 전 지점 목록에서 선택 또는 랜덤)
   - 스탬프 기록 생성: 적립 일시 + 지점명 + 순번
   - localStorage에 저장, 누적 개수 +1
3. [S2 적립 팝업] 자동 노출
   - 도장 찍기 애니메이션 재생
   - "스탬프가 적립되었어요!" + 지점명 + 날짜 + "N번째 스탬프"
4. (사용자) 닫기 버튼 또는 배경 탭
5. → [S1 메인 화면] 복귀
   - 스탬프 판에 새 스탬프가 채워진 상태
   - "최근 방문 지점" 영역이 방금 적립한 지점·일시로 갱신

예외 처리
- 스탬프 판(10칸)이 가득 찬 상태에서 적립 시: 데모에서는 새 판으로 넘어가며 누적 개수는 계속 증가 (보상 구조 확정 전 임시 정책)

---

## 3. 스탬프 내역 조회 플로우 (달력)

1. (사용자) 하단 탭 "스탬프 내역" 탭
2. → [S3 달력 화면] 《이번 달 달력 렌더링》
   - 적립이 있는 날짜에 빵 아이콘 마크 표시
   - 하단 요약: "이번 달 방문 N회 · 스탬프 N개"
3. (사용자) 마크가 있는 날짜 탭
4. 《해당 일 상세 노출》: 지점명 + 적립 시각 + 적립 개수
5. (사용자) 좌우 화살표로 이전/다음 달 이동
6. 《해당 월 데이터 렌더링》
   - 기록 없는 달: 빈 상태 문구 "이 달에는 방문 기록이 없어요"

예외 처리
- 마크 없는 날짜 탭 시: 상세 영역에 "이 날은 방문 기록이 없어요" 표시 (또는 무반응 — 개발 시 택1)

---

## 4. 닉네임 변경 플로우 (마이페이지)

1. (사용자) 하단 탭 "마이페이지" 탭
2. → [S4 마이페이지]
   - 현재 닉네임, 누적 스탬프 총 개수, 총 방문 횟수 표시
   - 로그인/계정 영역은 "준비 중" 표시
3. (사용자) 닉네임 변경 버튼 탭
4. [닉네임 수정 입력 상태] (사용자) 새 닉네임 입력
5. 《유효성 검사 (1~10자, 공백만 입력 불가)》
   - 통과 → 《localStorage 갱신》 → "변경되었어요" 확인 표시
   - 실패 → 오류 문구 노출, 저장 비활성화
6. → [S1 메인 화면] 이동 시 인사말에 새 닉네임 반영

---

## 5. 전체 플로우 다이어그램 (텍스트)

```
접속
 ├─ 닉네임 없음 → 닉네임 설정 → S1 메인
 └─ 닉네임 있음 ──────────────→ S1 메인
                                 │
        ┌────────────────────────┼────────────────────────┐
        ▼                        ▼                        ▼
  적립 트리거               탭: 스탬프 내역            탭: 마이페이지
        │                        │                        │
   《적립 처리》            S3 달력 조회              S4 닉네임 변경
        │                  (월 이동/일 상세)          (유효성 → 저장)
        ▼                        │                        │
  S2 적립 팝업                   └──────── 하단 탭 ───────┘
        │                              (상호 자유 이동)
   닫기 → S1 복귀
  (스탬프 판·최근 방문 갱신)
```

## 6. 상태별 화면 정리

| 화면 | 상태 | 표시 내용 |
|---|---|---|
| S1 메인 | 스탬프 0개 | 빈 스탬프 판 + "첫 스탬프를 찍어보세요" 안내 |
| S1 메인 | 적립 이력 있음 | 채워진 스탬프 판 + 최근 방문 지점·일시 |
| S3 달력 | 기록 있는 달 | 날짜 마크 + 월 요약 |
| S3 달력 | 기록 없는 달 | 빈 상태 문구 |
| S4 마이페이지 | 기본 | 닉네임 + 누적 요약 + 계정 "준비 중" |
